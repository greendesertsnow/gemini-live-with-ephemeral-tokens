WebView Authentication Test

This page demonstrates how React Native WebView can authenticate with the Next.js app.

Test Methods:

Instructions for React Native:


// Method 1: PostMessage (Recommended)
webViewRef.current?.postMessage(JSON.stringify({
  type: 'AUTH_TOKEN',
  token: 'your-pocketbase-token-here'
}));

// Method 2: Direct window method call
webViewRef.current?.injectJavaScript(`
  window.webViewAuth?.authenticate('your-token-here')
    .then(result => console.log('Auth result:', result));
  true;
`);
